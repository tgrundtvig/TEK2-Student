# TEK2 - Containerization and DevOps

Welcome to TEK2! This repository contains all student materials for the course.

## Course Overview

TEK2 teaches modern software development practices with a focus on:
- **Docker** containerization
- **Linux** command-line skills
- **CI/CD** automation with GitHub Actions
- **DevOps** practices

## Getting Started

### Prerequisites

- A computer with Windows 10/11, macOS, or Linux
- Administrator/sudo access to install software
- [Docker Desktop](https://www.docker.com/products/docker-desktop/) installed
- [Git](https://git-scm.com/) installed
- A [GitHub](https://github.com/) account

### Cloning This Repository

```bash
git clone https://github.com/tgrundtvig/TEK2-Student.git
cd TEK2-Student
```

### Updating Materials

Your instructor may update materials throughout the course. To get the latest:

```bash
cd TEK2-Student
git pull
```

## Weekly Structure

Each week folder contains:

```
week-XX-topic/
├── README.md           # Week overview and learning objectives
├── quick-reference.md  # Cheat sheet (print this!)
├── pre-class/          # Complete BEFORE class
│   ├── reading.md      # Conceptual introduction
│   └── exercises.md    # Hands-on preparation
├── class/              # Work through IN class
│   └── exercises.md    # Guided exercises
└── post-class/         # Complete AFTER class
    ├── advanced-tasks.md  # Deeper challenges
    └── hints.md           # Help if you get stuck
```

## Recommended Workflow

1. **Before class**: Read the pre-class materials and complete exercises
2. **During class**: Follow along with exercises, ask questions
3. **After class**: Complete advanced tasks to solidify learning

## Getting Help

- Check the `hints.md` file in each week's post-class folder
- Ask questions during class
- Review error messages carefully - they often explain the problem
